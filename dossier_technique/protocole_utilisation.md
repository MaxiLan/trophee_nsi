# Pré-recquis:
- python 3.0 ou plus
- pygame 2.3
    
# Protocole d'utilisation Linux/Windows:
- Executer le fichier python script.py (situé dans dossier_technique/sources/script.py) 
(Attention /!\ le fichier "particule.py" est nécessaire à l'exécution)
- Le programme vous demandera alors plusieurs informations dans la console python :
    - Rentrer la largeur de votre écran d'ordinateur en pixel dans la console python
    - Rentrer la hauteur de votre écran d'ordinateur en pixel dans la console python
    - Rentrer le nombre de particules que vous souhaitez afficher (généralement entre 50 et 75) dans la console python
- Une fenêtre s'ouvrira alors avec la simulation du nombre de particules souhaité
- Si vous bougez votre souris, les particules la suiveront
- Le clic-gauche vous permettra d'attirer toutes les particules vers la souris (susceptible de causer un crash)
- Le clic-droit donnera une nouvelle direction à toutes les particules
